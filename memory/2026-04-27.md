### clawpanel/ 
ClawPanel 是一个支持多 AI Agent 框架的可视化管理面板，目前支持 OpenClaw 和 Hermes Agent 双引擎，内置智慧 AI 助手。这是一个完整的前端项目（Vite + TypeScript + Tauri），提供多语言支持（包括繁体中文）。

### send_feishu.py
Python 脚本，用于向飞书机器人发送消息，支持：
- 命令行调用：`python send_feishu.py "消息内容" [--channel CHANNEL] [--template TEMPLATE] [--key KEY VALUE]...`
- 签名验证（HMAC-SHA256）
- 多种消息格式（文本/卡片）
- 模板变量替换
- 配置从 .env 和 .config/feishu/config.json 读取
